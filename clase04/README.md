## Flexbox

#### Para ver donde funciona

Se puede chequear en [Can I use](https://caniuse.com/?search=flexbox)

### ¿ Cómo lo aplico ?

El más importante es:

```CSS
.element {
  display: flex;
}
```

Otros son:

- ```flex-direction```

- ```flex-shrink```

- ```flex-wrap```

- ```justify-content```

- ```align-items```

- ```align-content```

Y en casa comando hay varias opciones para diferentes resultados.
